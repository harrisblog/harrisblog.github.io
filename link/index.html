友链添加须知

✔️ 含有本站友链

✔️ 网站长期能够正常访问

✔️ 最好是有自己的域名

✔️ 最好是定期写写文章叭

❌ 拒加涉黄、涉赌、涉毒、涉政网站

🖐 如果要下架本站，请在 留言信枫 备注好您的站点，好聚好散，后会无期~

💜 若你是 ♀，请无视以上所有要求 🙈

我会不定期访问友链，若友友们换域名了、服务器到期了或者不经营网站了麻烦给贵站所有友链打声招呼，长期无法访问贵站可能会被移除哟 QAQ（特别鸣谢友链除外）

本站友链信息

若贵站友链显示方式无需网站截图，可以将siteshot网站截图这一参数省略

友链添加方式

参照以下格式留言即可。如果你是 👇

建议先复制 👆 再来填写对应参数，若直接复制贵站预写好的友链信息，那么一定得注意友链信息变量名要与本站一致，因为不同站点所用到的变量名可能不同。

比如：网址导航，文档网站，CSDN，博客园等等

注意：

  1. 参数记得要与英文冒号之间空一格
  2. 个人搭建的博客网站我会放在本友链页，其他类型网站我会添加到我的网址导航页面
  3. 确保你的博客配置有sitemap.xml，因为本友链页会有爬虫来访问并爬取该站点文章进而生成朋友圈文章列表

站点截图方式 - siteshot

点击查看站点截图方式

建议使用 www.thum.io 提供的 API 以获取贵站的截图

方法一：先访问以下链接获取 API 提供的截图，再将图片上传至你的图床，最后将链接填入 siteshot

方法二：自行截图或用其它能代表贵站的图片，再将链接填入 siteshot。（要求：400✖266 像素 或 长宽比例 1.5）

方法三：直接省略（API 会自动截图，虽说 API 目前相当稳定，但本站不能保证以后也稳定）
